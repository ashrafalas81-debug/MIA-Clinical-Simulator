# MIA Clinical Simulator

## Project Description
MIA Clinical Simulator is a comprehensive platform designed to enhance clinical training by providing realistic clinical case scenarios.

## وصف المشروع
MIA Clinical Simulator هو منصة شاملة صممت لتحسين التدريب السريري من خلال تقديم سيناريوهات حالات سريرية واقعية.

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/ashrafalas81-debug/MIA-Clinical-Simulator.git
   ```
2. Navigate to the project directory:
   ```bash
   cd MIA-Clinical-Simulator
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
